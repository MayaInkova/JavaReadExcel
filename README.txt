Java програма, която прочита  Excel.
Допълнителни изисквания: Да се изведат само редове, които отговарят на определен
критерий (например "цена > 100 лв.").
Да се генерира отчет – средна стойност на числоваколона. 
Обработените данни да се запишат в нов Excel файл.





JavaReadExcel


Това е Java проект за работа с Excel файлове. Програмата чете данни от Excel файл,
 филтрира редове на базата на определен критерий (например, цена > 100 лв.), 
изчислява средната стойност на числова колона и записва резултатите в нов Excel файл.
