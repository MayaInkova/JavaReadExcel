JavaReadExcel


Това е Java проект за работа с Excel файлове. Програмата чете данни от Excel файл,
 филтрира редове на базата на определен критерий (например, цена > 100 лв.), 
изчислява средната стойност на числова колона и записва резултатите в нов Excel файл.

Използвала съм:
Java 
Apache POI библиотека за работа с E
